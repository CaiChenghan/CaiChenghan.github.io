hexo更换主题

  • 修改主题

    我们只需要在hexo的主题官网上挑选一个自己喜欢的主题，然后将其克隆到工程的thmes目录下，修改工程配置文件_config.yml，将theme修改成theme目录下的主题即可。

  • 主题配置

    hexo的主题配置，可参见其主题使用文档。
